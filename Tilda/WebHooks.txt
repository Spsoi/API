Аккаунт должен быть продлён.
Вкладки
  Мой сайт -> Редактировать сайт -> Кнопка настройки сайта -> Формы -> Другое -> WebHook
  
Если выдали через другой аккаунт доступ "Share"
  Если Аккаунт на платном тарифе и не продлён, весь функционал "Share" сайта будет заблокирован
    Нужно перейти на бесплатный и "Контент" будет доступен.

Если после нажатия кнопки в форме "выплывает" платёжная система.
  Мой сайт -> Редактировать сайт -> Кнопка настройки сайта -> Платёжные системы -> Подключенные платёжные системы -> "Выбрать из списка" -> Изменить -> Поле "ОТПРАВЛЯТЬ ДАННЫЕ В СЕРВИСЫ ПРИЕМА ДАННЫХ ТОЛЬКО ПОСЛЕ ОПЛАТЫ" -> снять галочку с "Да"
